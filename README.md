# sudoku
An command line sudoku game written in C.

The game includes an LP solver which is used to give guesses to the user, validate the board, and even solve the board fully.


The commands are: 
	* solve X
	* edit [X] (X is optional)
	* mark_errors X
	* print_board
	* set X Y Z
	* validate
	* guess X
	* generate X Y
	* undo
	* redo
	* save X
	* hint X Y
	* guess_hint X Y
	* num_solutions
	* autofill
	* reset
	* exit
